# Анализатор банковских выписок (Laravel + Docker)

Веб-приложение для автоматического парсинга, обработки и визуализации банковских выписок (Сбербанк и Т-Банк). 
Сейчас реализация обработки для файлов Т-Банк отсутствует. Смог достать актуальные выписки Сбербанк в pdf.
Формат excel получил из pdf на сайте [bank-statement-converter](https://nanonets.com/bank-statement-converter/#/)
Имея примеры других выписок, обработчик можно расширять.

Полученные файлы не сохраняются после обработки.

## Основные возможности
- **Загрузка файлов:** Поддержка форматов `.pdf`, `.xlsx`, `.xls`.
- **Аналитика:** Расчет общего оборота, доходов и расходов.
- **Визуализация:** 
  - График "Доходы vs Расходы" (Chart.js).
  - Диаграмма расходов по категориям (Chart.js).

## Технологии
- **Backend:** php:8.1-fpm, Laravel 12
- **Frontend:** Blade, Bootstrap 5, Chart.js
- **Парсинг:** `maatwebsite/excel`, `smalot/pdfparser`

## Установка и запуск
1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/ars-user-86867/bank-analyzer.git
   cd bank-analyzer
   ```

2. ***Настройка .env:***
  **Для теста необязательно**. 
  В корне проекта на основе файла ```.env.docker.example``` создать файл ```.env``` в котором указать данные для postgres.
  Иначе будут указаны данные по умолчанию
    ```
    DB_PORT: 5543
    DB_DATABASE: laravel
    DB_USERNAME: laravel
    DB_PASSWORD: secret
    ```

2. **Запуск приложения:**
    ```bash
    docker compose up -d
    ```


Приложение будет доступно по адресу: http://localhost:8010

Для остановки приложения

1. **Остановка приложения:**
    ```bash
    docker compose down
    ```
